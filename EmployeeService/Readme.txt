This example having functionality related to below ones.
1. Externalized Configuration
2. Profile management 
3. Actuator endpoint configuration.

Download the code base .

Eclipse --> File --> import --> Existing maven projects --> select the project code base.

Eclipse --> Run --> Run Configuration --> left side window select "Maven Build" --> Name ( employeeservice clean package)

Goals: clean package

Click on "Apply" button

Click on "Run" button.

For running the application open EmployeeServiceApplication.java class and right click -> Run As Spring Boot App

Open your favorite browser, paste the below  link and hit enter

http://localhost:8083/employee/registration

Note: Corresponding DB scripts available in resources folder.
